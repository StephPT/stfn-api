# STFN API

Api/Rest points for data used by applications on stun.

## Setup
To run the application, ensure a MySQL Server is setup with the correct tables. Then run the following Maven Commands:

Download Dependencies:
```bash
mvn clean install
```
Run Spring-boot:
```bash
mvn spring-boot:run
```
